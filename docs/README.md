# DexBot V2 Documentation Hub

DexBot V2 is a market-making and trading bot for the Blocknet XBridge protocol. It provides liquidity through automated
trading strategies.

## Key Features

- **Staggered Orders**: Place multiple orders at different price levels.
- **Dynamic Spreads**: Automatically adjust spreads based on inventory.
- **Safety Boundaries**: Circuit breakers to stop trading during extreme price movements.
- **Partial Order Management**: Support partial order fills.
- **Centralized Pricing**: Proxy server prevents API rate-limiting.
- **Auto-Recovery**: Resumes trading after crashes or network issues.
